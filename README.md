# Material Gojuon（五十音圖）


![](http://files.gracecode.com/2014_03_26/1395822850@320.png)


Material Gojuon（五十音圖）是一款 Material 风格的日语五十音助记软件。

其实市面上已经有很多的五十音图应用了，但是它们的设计以及界面等都感觉不是很理想，所以我考虑自己也写了一个类似这样的应用。

最后，恭祝学习日语进步 :^)
